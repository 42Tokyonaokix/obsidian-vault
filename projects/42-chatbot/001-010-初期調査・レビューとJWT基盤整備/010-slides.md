2026-03-23

---

## プロダクト概要

**エンタープライズ AI チャットボット** — 電力CS業務の自動化

- **ReAct エージェント**: LangGraph ベース、7ツール搭載
- **プラガブル RAG**: 4つの検索戦略を切替可能
- **マルチエージェント**: Data / Action / Knowledge の3エージェント
- **リアルタイム応答**: SSE ストリーミング
- **セキュリティ**: JWT RS256 / 入出力ガードレール

技術: Python 3.12 / FastAPI / PostgreSQL + pgvector / React 19
