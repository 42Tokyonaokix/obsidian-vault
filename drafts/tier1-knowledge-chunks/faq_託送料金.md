## チャンク一覧

### Q1: 託送料金とは何ですか？

**Q**: 託送料金とは何ですか？電気料金にどのように含まれていますか？

**A**: 託送料金（接続送電サービス料金）は、一般送配電事業者（東京電力パワーグリッド、関西電力送配電など）が所有・運営する送配電ネットワークの利用料金です。

- **基本料金**: 契約電力（kW）に基づく固定費部分。計算式は「基本料金単価（円/kW）× 契約電力（kW）× (185 - 力率) / 100」
- **従量料金**: 使用電力量（kWh）に基づく変動費部分。時間帯別（昼間・夜間）や季節（夏季・その他季）で単価が異なる場合があります。

託送料金の単価はエリア（送配電事業者）ごとに異なり、各送配電事業者の託送供給等約款に基づいて設定されています。DGMの `consignment_monthly_charges` API で託送料金の明細を確認できます。

### Q2: 基本料金や従量料金の単価が変わったのはなぜですか？

**Q**: 基本料金単価が10月分から変更になりました。従量料金単価の変更も託送料金変更によるものですか？

**A**: 基本料金や従量料金の単価変更は、主に以下の理由で発生します。

- **託送料金の改定**: 一般送配電事業者が託送供給等約款を改定した場合、託送料金部分が変更されます。これにより、お客さまの基本料金・従量料金の単価が変動する場合があります。
- **レベニューキャップ制度**: 2023年度から導入されたレベニューキャップ制度により、送配電事業者の収入上限が定期的に見直されるため、託送料金も更新されます。

具体的にどの料金構成要素が変更されたかは、DGMの請求明細でご確認いただけます。ご不明な点はカスタマーサポート（customer_support@digitalgrid.com）までお問い合わせください。

### Q3: 送配電事業者の託送料金見直しは自社の基本料金に影響しますか？

**Q**: 東京電力の託送基本料金の見直しが自社の基本料金に関係するかを教えてください。

**A**: はい、お客さまの需要場所が当該送配電事業者の供給区域内にある場合、託送料金の見直しは電気料金に影響します。

- **影響の仕組み**: お客さまの電気料金のうち、託送料金部分は送配電事業者の約款に基づく単価で算定されます。送配電事業者が託送料金を改定した場合、その分がお客さまの電気料金に反映されます。
- **影響範囲**: 影響を受けるのは、当該送配電事業者の管轄エリアに需要場所が存するお客さまのみです。他エリアの拠点には影響しません。

託送料金の改定情報はDGMでの請求明細に反映されます。事前の見積もりをご希望の場合は、担当営業またはカスタマーサポートまでお問い合わせください。
